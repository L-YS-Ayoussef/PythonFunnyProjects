# PythonFunnyProjects 🎉  
A Collection of Fun Python Mini-Apps

## 🌍 Overview  
**PythonFunnyProjects** is a collection of small, playful Python projects built mainly for learning, experimentation, and fun.  
Each project lives in its own folder with its own `README.md`, code, and assets.

This repo is a great place to explore beginner–intermediate Python concepts through quirky little apps.

---

## 🚀 Projects

### 1️⃣ Pomodoro ⏱️ – Focus Timer  
**Folder:** `PomodoroTimer/`  
A colorful desktop Pomodoro timer built with Tkinter and Pygame.  
Features work/break cycles, start–pause–resume controls, an alarm sound, and visual checkmarks with total time worked.

### 2️⃣ Quizzer-True-False-Quiz-App ✅❌
**Folder:** `Quizzer-True-False-Quiz-App/`  
Quizzer is a small desktop quiz application built with Python and Tkinter.
It asks a series of true/false questions, updates your score as you answer, and shows a final message when you’ve completed the quiz.
